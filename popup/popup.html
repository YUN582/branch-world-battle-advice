📦 업데이트 방법

방법 1: Git Pull (추천)
  1. 확장 프로그램 폴더를 파일 탐색기에서 엽니다
  2. 주소창에 cmd 를 입력하고 Enter
  3. git pull 입력 후 Enter
  4. 크롬 주소창에 chrome://extensions 입력
  5. 이 확장 프로그램의 🔄 새로고침 버튼 클릭
방법 2: 자동 스크립트
  1. 확장 프로그램 폴더의 update.bat 파일을 더블클릭
  2. 크롬 주소창에 chrome://extensions 입력
  3. 이 확장 프로그램의 🔄 새로고침 버튼 클릭
새 채팅 메시지가 도착하면 채팅 로그를 자동으로 맨 아래로 스크롤합니다

코코포리아 음량 슬라이더를 렉 없는 경량 버전으로 교체합니다
콘솔에 상세 디버그 로그를 출력합니다 (F12 개발자 도구)
